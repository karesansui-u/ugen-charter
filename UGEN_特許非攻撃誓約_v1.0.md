# UGEN 特許非攻撃誓約（Patent Non-Assertion Pledge）

本文書は、UGEN特許群の使用に関する非攻撃誓約を定める。
存続論の普及と人類全体の存続力向上を目的とし、条件を遵守する使用者に対しては特許権を行使しない。

---

## 1. 誓約の趣旨

### 1.1 背景

特許権は本来、発明者の保護を目的とする。しかし、存続論は「人類の存続」という目標を持つ。
この目標は、技術の独占ではなく普及によって達成される。

同時に、存続論の核心原則が歪められた形での利用は、目的に反する。
したがって、**条件付きの開放**という設計を採用する。

### 1.2 Tesla誓約との違い

| 観点 | Tesla方式 | UGEN方式 |
|------|-----------|----------|
| 条件 | 「善意（good faith）」| 明示的な原則遵守 |
| 相互主義 | 暗黙の期待 | 明示的な条件 |
| 目的 | 市場拡大 | 存続力向上 |
| 違反時 | 誓約撤回 | 誓約撤回 + 原則復帰要求 |

### 1.3 中立性

本誓約は特定の国家・政治体制・イデオロギーを対象としない。
適用は行為（攻撃の有無）に基づき、主体の性質には基づかない。

---

## 2. 非攻撃誓約

### 2.1 誓約内容

本誓約の宣言主体である株式会社テンクック、および代表取締役個人（対象特許の出願人）は、以下の条件を**すべて**満たす使用者に対し、特許権に基づく訴訟その他の権利行使を行わないことを誓約する。

> **注**: 一般社団法人UGEN設立後は、本誓約は法人に承継され、「UGEN特許群の権利者（継承組織およびその継承者）」が誓約主体となる。

### 2.2 誓約が適用される条件

以下のすべてを満たす場合、本誓約が適用される：

1. **技術提供の平等性**
   - 技術を特定の集団に限定せず、すべての人に平等に提供する
   - 富裕層優遇、特定国家優遇など差別的アクセスを行わない

2. **侵略・抑圧目的での不使用**
   - 侵略目的での使用を行わない
   - 人間の尊厳を侵害する目的で使用しない
   - 存続係数を「低い者を排除する」目的で使用しない

3. **余白の保護**
   - 対象の選択肢（N_eff）、余白（μ）を減少させる目的で使用しない
   - 「最適化」の名のもとに多様性を破壊しない

4. **停止権の尊重**
   - 対象者に離脱・停止の権利を保証する
   - 強制的な参加を要求しない

5. **相互非攻撃**
   - UGEN関連特許に対し、無効審判・侵害訴訟等の攻撃を行わない
   - 他のUGEN誓約遵守者に対し、特許攻撃を行わない

6. **透明性**
   - UGEN技術の使用について、合理的な範囲で情報開示を行う

7. **改良の還元（Reciprocity）**
   - 本特許群を利用して新たな特許を取得した場合、その特許についても本誓約と同様の条件（非攻撃誓約）を適用することを約束する
   - これにより、UGENエコシステム内での特許独占を防ぎ、相互利用を促進する

---

## 3. 誓約が適用されない場合

以下のいずれかに該当する場合、本誓約は適用されず、通常の特許権行使が可能となる：

### 3.1 直接的な条件違反

- 上記2.2の条件（1〜7）のいずれかに違反した場合
- 違反が是正されれば、誓約は再適用される

### 3.2 攻撃的使用

- UGEN特許群に対する無効審判、侵害訴訟等を提起した場合
- 他のUGEN誓約遵守者に対し特許攻撃を行った場合
- この場合、誓約は即座に失効する

### 3.3 悪意ある使用

- 存続係数を操作し、特定の主体を不当に排除する目的で使用した場合
- 「UGEN認証」を詐称し、実際には原則に反する運用を行った場合
- 存続論を歪曲し、差別・排除を正当化する目的で使用した場合
- **政治的宣伝・世論操作（Propaganda）：**
  - 特定の政治的勢力、宗教、国家、または個人の権威付けのために、存続論の指標（S値など）を恣意的に利用、または改ざんして流布する行為
  - AIによる大量の自動生成等を用いて、人為的に合意形成を歪める行為（アストロターフィング）
- **偽装攻撃（False Flag）：** 他者を陥れる目的で、他者になりすまして攻撃を行う、または他者のシステムを乗っ取って攻撃を行わせる行為
  - この事実が判明した場合、なりすまされた被害者の権利は保護され、**実行者およびその関連主体は即座に永久追放（復帰不可）**となる
- **ガバナンス攻撃：**
  - UGEN法人の定款・規則を不正に変更し、憲章の核心（独占禁止・停止権・余白保護）を骨抜きにしようとする行為
  - 法人の意思決定機関（社員総会・理事会・評議員会等）を乗っ取り、憲章に反する決議を行わせる行為
  - 脅迫・買収・詐欺等により、法人の役員・社員・受託者に憲章違反の行為を行わせる行為
  - この行為には、直接の実行者だけでなく、教唆者・共謀者・資金提供者も含まれる

### 3.4 関連主体への波及（連座条項）

攻撃主体と関連する以下の主体についても、誓約は適用されない：

**企業グループの場合：**
- 攻撃主体と実質的に協調して行動する法人（資本関係の有無・比率を問わない）
- 攻撃主体の親会社・子会社・関連会社
- 攻撃主体と共通の支配者を持つ法人
- 攻撃行為を共同で行った、または資金・技術を提供した主体

> **注**: 関連性の判定において、議決権比率は参考値に過ぎない。形式的に独立であっても、実質的に協調して行動している場合は同一主体とみなす。N_eff（実効的な独立選択肢数）の観点から、「独立した競争関係にあるか否か」が判定基準となる。

**公的機関介入の場合：**
- 攻撃を指示・支援した公的機関
- 当該機関が支配する公営企業・関連機関
- 攻撃に協力した管轄区域内の民間企業（協力の事実が確認された場合のみ）

> **備考：** 「公的機関」には国家（政府機関）、自治体（地方政府）、国際機関等を含む。

**目的：**

```
従来：
  子会社が攻撃 → 子会社のみ失効 → 親会社は使い続けられる

連座条項：
  子会社が攻撃 → グループ全体失効 → 親会社が子会社を止めるインセンティブ
```

公的機関介入の場合も、当該管轄区域の企業が「当局に攻撃をやめさせる」圧力をかけるインセンティブが生まれる。

**例外（善意の第三者保護）：**
- 攻撃行為を知らず、かつ合理的に知り得なかった関連主体
- 攻撃行為に反対し、公に離脱を表明した関連主体
- 攻撃後に資本関係を解消した旧関連主体（解消から1年経過後）

**復帰条件：**
- 攻撃主体が攻撃を取り下げ、是正した場合
- 関連主体が攻撃主体との関係を解消した場合
- 復帰には、UGEN連合（または権利者）の確認を要する

**組織再編・継承時の適用（継承）：**
攻撃主体の事業、資産、または支配権を継承した組織（新設分割、合併、事業譲渡等による後継組織を含む）に対しても、本誓約の失効は継承される。
看板を掛け替えても、攻撃の事実は消滅せず、特許使用権は復活しない。

### 3.5 共謀・教唆への対処

攻撃行為を直接行わなくても、以下に該当する者は攻撃主体と同等に扱う：

**教唆者（依頼者）：**
- 攻撃を指示・依頼した者
- 攻撃の報酬を提供した者
- 攻撃を行うよう誘導・説得した者

**共謀者：**
- 攻撃の計画に参加した者
- 攻撃に必要な情報を意図的に提供した者
- 攻撃の成功から利益を得る合意があった者

**資金提供者：**
- 攻撃のための資金を提供した者
- 攻撃者の訴訟費用を負担した者
- 攻撃者に対し、攻撃を条件とした融資・投資を行った者

**処理：**
- 教唆者・共謀者・資金提供者と実行者の関係が判明した時点で、**全員を同時に失効対象**とする
- 一方のみが判明した場合でも、その者を失効対象とし、他方の調査を継続する
- 関係の証明は、状況証拠を含む合理的な証拠で足りる

**公的機関による教唆：**
公的機関が攻撃を指示・依頼した場合、以下のすべてが失効対象となる：
- 指示を出した公的機関
- 実行者（個人・企業）
- 当該公的機関と同一管轄のすべての公的機関
- 当該公的機関が支配する公営企業
- 当該管轄の中央銀行相当機関・公的金融機関

> **備考：** 「公的機関」には国家（政府機関）、自治体（地方政府）、国際機関等を含む。

**例1（個人・企業による教唆）：**
```
富裕層A が 個人B に攻撃を依頼
        ↓
両者の関係が判明
        ↓
A（依頼者）と B（実行者）両方の誓約が即座に失効

さらに：
- Aの関連企業（3.4適用）
- Bに資金提供した者
すべてが調査・失効対象
```

**例2（公的機関による教唆）：**
```
管轄Xの公的機関 が 企業B に攻撃を依頼
        ↓
関係が判明
        ↓
即座に失効：
- 指示を出した公的機関
- 企業B
- 管轄Xの全公的機関
- 管轄Xの全公営企業
- 管轄Xの中央銀行相当機関

さらに：
- Bの関連企業（3.4適用）
- 管轄Xと協力した民間企業
すべてが調査・失効対象
```

**管轄配下全体への波及の目的：**
```
公的機関が攻撃を指示した場合：
  → 当該管轄全体がUGEN特許群を使えなくなる
  → 管轄内企業・機関が「当局に攻撃をやめさせる」圧力をかける
  → 当局内部でも「攻撃は割に合わない」という認識が広がる
```

**民間企業への影響：**
- 公的機関が教唆した場合でも、当該管轄の**民間企業**は直ちには失効対象としない
- ただし、攻撃に協力した民間企業は失効対象（3.4適用）
- 民間企業が当局に圧力をかけるインセンティブを維持するため

### 3.6 疑義がある場合の対処（立証責任の転換）

証拠が不十分だが関与の疑いがある場合の手続きを定める。

**疑義の基準：**
攻撃主体との関連性があり、かつ攻撃との近接性がある場合、関与の疑いがあるとみなす。

- **関連性**：資本関係、人的関係、取引関係、その他の実質的つながり
- **近接性**：攻撃に時期的・内容的に近接した協力、資金・技術・情報の提供

**暫定停止：**
- 疑義がある主体に対しては、誓約の適用を暫定的に停止する
- 暫定停止は、疑義の通知から即座に効力を発する
- 暫定停止中は、新規の特許使用を開始できない

**立証責任の転換：**
- 疑義がある主体は、関与がないことを自ら証明する責任を負う
- 証明すべき内容：
  - 攻撃の意思決定に関与していないこと
  - 攻撃のための資金・技術・情報を提供していないこと
  - 攻撃から利益を得る立場にないこと
- 証明期間：通知から60日
- 証明方法：書面、第三者監査、その他合理的な証拠

**判定結果：**
- 証明できた場合 → 誓約再適用、暫定停止解除
- 証明できなかった場合 → 失効確定（3.4の連座条項適用）
- 虚偽の証明を行った場合 → 失効確定 + 信頼性の永久喪失

**被害者の救済（なりすまし・乗っ取り）：**
- 攻撃が自身の意思に基づかないこと（ハッキング被害、名義冒用、偽装工作等）を合理的に疎明できた場合、特許使用権の停止は行わない、または即座に解除する。
- 調査期間中であっても、明らかに被害者である可能性が高いと判断される場合は、暫定的な使用継続を認める措置（Grace Period）を適用する。

**実質支配者への遡及：**
- ペーパーカンパニー、SPC等を通じた責任回避は認めない
- 名目上の株主ではなく、実質支配者（beneficial owner）を基準とする
- 複数のダミー会社を経由している場合も、最終的な受益者まで遡及する

**目的：**
```
従来の抜け穴：
  攻撃 → ダミー会社を使う → 証拠残らない → 本体は使い続けられる

立証責任転換：
  攻撃 → 疑われる → 潔白を証明できない → 失効
```

「証拠がないから無罪」ではなく、「疑われたら潔白を証明する」設計。

### 3.7 透明性の原則

すべての判定プロセスにおいて、透明性を確保する。

**権利者の公開義務：**
疑義・失効の判定を行う際、権利者は以下を公開しなければならない：
- 疑義を抱くに至った理由・証拠
- 関連性を判断した根拠
- 適用した条項（3.2〜3.6のいずれか）
- 判定のプロセスと経緯

**被疑主体の公開義務：**
疑義をかけられた主体は、以下を公開しなければならない：
- 潔白を証明するために提出した証拠
- 反論の内容
- 第三者監査の結果（実施した場合）

**争点の公開：**
- 判定において争点となった事項はすべて公開する
- 双方の主張を対照可能な形で公開する
- 最終判定の理由を詳細に公開する

**公開の方法：**
- UGEN連合の公式チャネル（ウェブサイト、リポジトリ等）で公開
- 公開から少なくとも10年間はアクセス可能な状態を維持
- 改竄防止のため、ハッシュ値を記録

**公開の例外：**
以下の情報は、正当な理由がある場合に限り非公開とできる：
- 個人のプライバシーに関わる情報（氏名等は匿名化可能）
- 営業秘密に該当する技術情報
- 進行中の法的手続きに影響を与える情報

ただし、例外を適用する場合も、その理由を公開する。

**目的：**
```
透明性がない場合：
  権利者が恣意的に判定 → 不信感 → 誓約の信頼性低下

透明性がある場合：
  判定プロセスが検証可能 → 公正性の担保 → 誓約の信頼性向上
  他者も先例を参照可能 → 予測可能性の向上
```

**相互監視効果：**
- 権利者も「不当な疑義をかけた」と批判されるリスクを負う
- 被疑主体も「虚偽の証明をした」と公開されるリスクを負う
- 双方に誠実な対応を促すインセンティブ

### 3.8 派生物独占（Derived Work Monopolization）

UGENプロトコルを利用して開発・発見された成果物を独占し、他者の選択肢（N_eff）を著しく減少させる行為は、憲章の核心1（独占禁止）に違反し、本誓約の失効事由となる。

**適用範囲：**
- UGENの診断・予測・最適化機能を利用して得られた成果物
- 医薬品、治療法、技術、知見、アルゴリズム等を含む
- 派生の連鎖（UGEN → 発見A → 発見B）にも適用される

**違反の例：**
- 必須医薬品の独占的価格設定（代替なき高額設定により患者の選択肢を消滅）
- 研究成果の不当な囲い込み（科学の進歩を阻害）
- 代替治療法・代替技術へのアクセス妨害
- 特許プールへの参加拒否による市場閉鎖

**執行メカニズム：**
```
派生物独占の検出
    ↓
違反者の N_eff 項が減少（信頼資本の毀損）
    ↓
S = N_eff × (μ/μ_c) × exp(-δ) が低下
    ↓
S < S_c に至った場合、本誓約が失効
    ↓
UGEN特許群の使用権喪失
```

**正当な利益の保護：**
- 合理的な期間・価格での特許保護は許容される
- 問題は「選択肢の消滅」であり「利益の獲得」ではない
- ジェネリック薬の参入妨害、強制実施権への抵抗等は違反に該当しうる

**科学の進歩との関係：**
- 本条項は、UGENを使って科学を進歩させた成果を独占し、さらなる進歩を妨げる行為を禁止する
- 「発見の独占」は、発見者自身の存続係数を低下させる自己矛盾的行為である
- 科学は累積的であり、独占は累積を止める

**独占の数学的定義：**

独占は N_eff < 2（実効選択肢数が2未満）として定義される。これは恣意的判断ではなく、数値による客観的判定である。

```
N_eff = 1  → 完全独占（選択肢なし）→ 違反
N_eff = 2  → 複占（最小限の競争）→ 警告水準
N_eff ≥ 3  → 健全な競争 → 許容
```

**独立性の判定基準：**

N_eff のカウントにおいて、以下のいずれかに該当する主体は「同一」とみなす:

| 判定項目 | 同一とみなす条件 | 理由 |
|---------|-----------------|------|
| 実質的協調 | 協調して行動している事実（資本関係の有無を問わない） | **本質的判定基準** |
| 資本関係 | 親子会社・関連会社関係（比率は参考値） | 意思決定の統一の可能性 |
| 国家統制 | 同一国家の実質的指揮下 | 独立した競争が存在しない |
| 協調行動 | カルテル・価格協定・市場分割 | 競争の排除 |
| 技術依存 | 同一の必須特許・基盤技術に依存 | 上流での独占 |
| 人的支配 | 経営陣の実質的共通 | 実質的な単一主体 |

> **原則**: 議決権比率（20%等）は法務上の参考値であり、判定の決定要因ではない。真の判定基準は「独立した競争関係にあるか否か」である。形式的に独立していても、実質的に協調していれば N_eff = 1 とみなす。

**測定対象の限定：**

N_eff は「当該市場・技術領域」に限定して測定する。

```
例：薬Xの独占判定
  → 薬Xの代替品を提供できる組織のみカウント
  → 他領域の企業は計算に含まない
  → 無関係な組織の巻き込みを防止
```

**物理的帰結としての性質：**

この判定は裁判所の判断を必要としない。N_eff は観測可能な数値であり、独占の有無は数学的に決定される。

### 3.9 システム防衛の物理的帰結（System Defense Consequence）

UGEN憲章第12条の2（存続阻害要因に対する物理的帰結）に基づき、システムが参加者を「高エントロピー源」と判定し、提供リソースを「生存維持レベル（Life Support Level）」へ自動縮退させた場合、これはプロトコルの物理的挙動であり、権利者による特許権の行使または誓約違反には該当しない。

- **自動性の確認**: この縮退は、脅威の検知に伴う自動的な相転移であり、人為的な制裁措置ではない。
- **復帰の条件**: 制限の解除は、期間の経過ではなく、脅威の除去と信頼スコア（Trust Metric）の回復によってのみ物理的に可能となる。
- **免責**: この機能発動によって生じた、攻撃的・拡張的活動の停滞、およびそれに伴う経済的損失について、権利者は一切の責任を負わない。ただし、生存維持（医療、基本インフラ等）に必要な最低限のリソースは維持される。

## 4. 法的効果

### 4.1 誓約の性質

- 本誓約は法的拘束力のある約束（covenant not to sue）である
- 誓約は特許権の放棄ではない

### 4.2 誓約の継承

- 本誓約は特許権の移転に伴い、移転先にも継承される
- UGEN特許譲渡条件（別文書）に基づく譲渡においても、本誓約は維持される

### 4.3 適用範囲

本誓約は以下の特許に適用される：

- UGEN存続係数算出に関する特許
- UGEN認証に関する特許
- 存続論に基づく評価・予測システムに関する特許
- その他、権利者がUGEN特許群として指定した特許

なお、本誓約の対象となる「UGEN技術」は、その原理（L1）が物理法則に基づくため、ソフトウェアや情報処理システムに限定されない。以下の全領域を含むものとする：

1.  **情報空間**: AI、OS、組織管理、経済システム
2.  **生物空間**: 医療、バイオテクノロジー、細胞制御、農業
3.  **物理空間**: エネルギー、気象制御、都市インフラ、ロボティクス

これら全ての領域において、UGENの原理を利用する主体は、本誓約の保護を受け、同時にその制約（憲章遵守）に服する。

### 4.4 技術的自己執行性（存続係数との連動）

本誓約の効力は、法的訴訟のみならず、UGENプロトコルの技術的仕様（存続係数 S）によっても担保される。

- **信頼の喪失と係数低下**: 本誓約への違反行為は、UGENプロトコルにおいて「信頼資本（$N_{eff}$）の毀損」として観測される。
- **物理的帰結としての停止**: これにより対象主体の存続係数 $S$ が臨界値 $S_c$ を下回った場合、C群（C5: 資源依存型生存維持等）の機能により、プロトコルの利用は技術的に不可能となる。
- **恣意性の排除**: この停止措置は権利者の恣意的な判断ではなく、公開された数式と観測データに基づく物理的・数学的な帰結として実行される。$S_c$ の決定根拠はパーコレーション理論に基づく（詳細は憲章付録D参照）。

```
誓約失効のプロセス：
  違反行為 → N_eff激減 → S < S_c（臨界割れ）
           → C5（トークン枯渇）発動 → システム停止
```

**裁判所に依存しない理由：**
- 停止は数式（自然法則）の帰結であり、法的判断（人為）を待たない
- 復旧するには、信頼（$N_{eff}$）を回復し、数式上で $S > S_c$ を満たす必要がある

### 4.5 誓約の不可逆性

本誓約は、創設者の死亡・不在・意思表示不能によっても撤回されず、永続的に効力を維持する。

- **撤回不能**: 本誓約は一度公開された後、いかなる理由によっても撤回できない
- **継承の強制**: 特許権の相続人・承継人は、本誓約を承継する義務を負う
- **攻撃の無意味性**: 創設者への攻撃は、憲章第13条により攻撃者の永久使用停止を招くのみであり、誓約の撤回や特許の独占には繋がらない

## 8. FAQ

### Q1: 誓約を利用するために登録は必要ですか？

不要です。条件を満たしていれば自動的に誓約が適用されます。
ただし、後の紛争を避けるため、利用開始時に書面で通知することを推奨します。

### Q2: 営利目的で使用できますか？

可能です。営利・非営利を問わず、条件を満たせば使用できます。
ただし、料金設定により事実上のアクセス制限が生じる場合は、条件1（平等性）に抵触する可能性があります。

### Q3: 既存の特許を持っている場合はどうなりますか？

本誓約は、使用者が自身の特許でUGEN関係者を攻撃しないことを条件としています（条件5）。
既存特許の保有自体は問題ありませんが、それを使った攻撃は誓約失効の原因となります。

### Q4: 競合他社も条件を守れば使えるのですか？

はい。条件を守る限り、競合他社であっても使用可能です。
これは存続論の普及を優先する設計です。

### Q5: 私の知らないところで子会社や配下の組織が勝手に攻撃したら、私も巻き添えですか？

**はい、再帰的な「物理的帰結」として、あなた自身の特許使用権も自動的に失効します。**
これは「トカゲの尻尾切り」による攻撃を防ぐための、プロトコル上の不可避な動作です。

この失効は、運営者の「裁定」によるものではなく、攻撃イベントが検知された際に依存関係（資本・組織構造）を遡って波及する自動的なプロセスです。重力が対象を選ばないのと同様、こちら側で個別に制御したり、停止を止めたりすることは不可能です。

そのため、UGENを利用する組織・国家は、自らのガバナンスが及ぶ範囲（子会社、委託先、配下の機関、軍部など）を「自らの身体の一部」として管理する責任を負います。内部統制の不備は、システム上は「致命的な構造的脆弱性」とみなされ、不可避な機能停止を招くことになります。
