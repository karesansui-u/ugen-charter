# UGEN Charter (Canonical Release)

UGEN は、**組織や社会の存続** を支援する世界共通プロトコルである。

UGEN は OSS として世界に開かれるが、**思想（Charter）だけは不可逆にロックされる。**

本憲章はプロトコル群の追加・拡張に対して常に適用され、個別の構成や公開順序は運用規程に委任される。

## 前文（Preamble）

存続には、選択肢・整合性・余白が必要である。

我々は「何が正義か」を定義しない。正義は立場によって異なり、争いの火種となる。

**人種・宗教・民族文化の対立に関与しない。** どの立場も、その角度から見れば合理的な理由がある。対立の多くは、翻訳の問題、歴史解釈の問題、認知の問題である。我々はどちらが正しいかを判断せず、対話の基盤だけを提供する。

**宗教や政治の主体にならない。** UGENは信仰でも政党でもない。もし誰かがUGENを宗教運動や政治運動に利用しても、それはUGENの行為ではない。UGENはプロトコル・インフラであり、運動ではない。

**信仰の領域に立ち入らない。** 物理法則の解釈は各自に委ねる。

代わりに我々は、生命と文明に共通する **存続** という事実だけを見つめる。**物理法則と存続の原理に従う。** それだけが、すべての立場を超えて共有できる土台である。

UGEN は、個人でも国家でも企業でもなく、**構造そのものが真実を語る** という原理に基づく。

本憲章は、UGEN プロトコル群が特定の技術、国家、企業、または勢力の影響下に置かれない **中立の信用基盤** であり続けるために、最低限必要な約束を記す。

## 核心（変えられないもの）

以下の3つは、UGENの存在根拠であり、いかなる改訂によっても変更されない。

### 1. 独占禁止

N_eff（実効選択肢数）を最大化するため、単一主体による支配を許さない。

**補足**:
- これは反資本主義ではない。企業が利益を得ることは問題ない。
- 問題は「一者が全体を支配する構造」であり、それは熱力学的に脆弱である。
- 競争は許容される。独占による選択肢の消滅が禁止される。
- 複数の実装、複数の運営主体が共存する状態を正常とする。

**派生物独占禁止**:

UGENプロトコルを利用して開発・発見された成果物（医薬品、治療法、技術、知見等）を独占的に保有し、他者の N_eff（選択肢数）を著しく減少させる行為は、本核心に違反する。

- **適用範囲**: UGEN の診断・予測・最適化機能を利用して得られた成果物
- **違反の例**:
  - 必須医薬品の独占的価格設定（代替なき高額設定）
  - 研究成果の不当な囲い込み（科学の進歩を阻害）
  - 代替治療法・代替技術へのアクセス妨害
- **執行メカニズム**: 違反者の存続係数 S における N_eff 項が減少し、S < S_c に至った場合、プロトコル使用権が失効する
- **正当な利益の保護**: 合理的な期間・価格での特許保護は許容される。問題は「選択肢の消滅」であり「利益の獲得」ではない

**独占の数学的定義**:

独占は N_eff < 2（実効選択肢数が2未満）として定義される。これは恣意的判断ではなく、数値による客観的判定である。

```
N_eff = 1  → 完全独占（選択肢なし）
N_eff = 2  → 複占（最小限の競争）
N_eff ≥ 3  → 健全な競争
```

**独立性の判定基準**:

N_eff のカウントにおいて、以下のいずれかに該当する主体は「同一」とみなし、+1 しない:

| 判定項目 | 同一とみなす条件 |
|---------|-----------------|
| 資本関係 | 議決権 20% 以上の共通支配 |
| 国家統制 | 同一国家の実質的指揮下にある |
| 協調行動 | カルテル・価格協定・市場分割協定 |
| 技術依存 | 同一の必須特許・基盤技術に依存 |
| 人的支配 | 経営陣の実質的共通 |

**測定対象の限定**:

N_eff は「当該市場・技術領域」に限定して測定する。無関係な組織は独占判定に影響しない。

### 2. 停止権

存続係数 S < S_c（臨界値）のとき、当該処理は自動停止する。

**補足**:
- これは検閲ではない。内容を判断して止めるのではない。
- 「構造的に存続不可能」と数式で判定された場合のみ発動する。
- 判定ロジックはオープンソースで公開され、誰でも検証・反証できる。
- 停止は「罰」ではなく「物理的帰結」である。重力で物が落ちるのと同じ。
- **停止の対象はUGENプロトコルの使用権である。** エネルギー供給そのものは各国・各主体の責任であり、UGENが直接制御するものではない。
- **復旧も物理的帰結である。** S >= S_c を証明すれば、人間の「承認」なしに自動復旧する。詳細は第14条を参照。

### 3. 余白保護

μ（余白）を奪う行為は、存続を脅かすものとして扱う。

**補足**:
- これは非効率の推奨ではない。
- 「効率100%」は変化に対応できない硬直状態であり、環境変化で全滅する。
- 余白とは、選択肢があること、逃げ場があること、遊びがあること。
- 具体例: 強制参加の禁止、離脱権の保障、秘匿の権利。
- 余白の量は測定可能であり、恣意的な判断ではない。

## 1. 基本原則

### **第1条：尊厳と自由**

人間が単なるデータやリソースとして扱われてはならない。

監視や強制によらず、自らの意志で選択し、行動する権利を守る。

### **第2条：構造的透明性**

なぜそう判断したのか、誰でも検証できなければならない。

ブラックボックスによる不利益な処分は認めない。

*   **検証可能性**: ロジック、データソース、判定基準を開示する。
*   **公平性の担保**: 選択肢の提示には、物理乱数等の偏りのない手段を用いる。
*   **不完全性の受容**: すべての保証は確率的である。誤りがありうることを前提に、異議申し立て手段を確保する。

### **第3条：誰でも使える、いつでも抜けられる**

誰もが参加でき、誰もがいつでも離脱できる。

強制参加は許されない。独占も許されない。

*   **アクセス権**: 国・企業・文化を問わず、誰でも利用できる。
*   **ローカル主権**: 各地域の法規制や文化を尊重する。
*   **離脱権**: いつでも不利益なく、自分のデータを持ち出して離脱できる。
*   **強制禁止**: 本プロトコルの利用を、生活基盤や権利の条件にしてはならない。

### **第4条：余白を守る**

効率を極限まで追求すれば、システムは死ぬ。

我々は意図的に「無駄」と「遊び」を守る。

*   **秘する権利**: すべての主体は、自らの内面を隠す権利を持つ。
*   **ゆらぎの実装**: 完全な管理社会を防ぐため、意図的な非効率やランダム性を組み込む。
*   **美**: 余白は無駄ではない。創造性と尊厳が宿る聖域である。

### **第5条：人間中心の制御権**

どれだけ技術が進化しても、最終的な意思決定権と停止ボタンは人間が持つ。

AIやシステムが賢くなっても、人間が「止める」と言えば止まる。この約束は変えない。

*   **物理的遮断**: 暴走したシステムを、人間が物理的に止められる手段を常に確保する。
*   **支配の防止**: 人間が形式上は判断しているが、実質的にはAIの言いなりになっていないかを監視する。
*   **能力の維持**: 人間は、システムを監視・制御する能力を維持する責務を負う。

## 2. 禁止事項

### **第6条：攻撃・戦争利用の禁止**

UGEN を人を殺すために使ってはならない。

*   致死性自律兵器の最終意思決定に使用してはならない。
*   人間の行動誘導・思想統制に用いてはならない。
*   政治的弾圧・特定集団への差別に利用してはならない。

**UGENは軍事判断をしない。** 防衛か侵略かの判断は行わない。エネルギーを使って何をするかは各国・各主体の責任である。ただし、UGENを直接殺傷に使うことは禁止される。

**存続係数による停止は発動する。** 政治的判断はしないが、物理的判断はする。いかなる行為であれ、人類全体の存続係数を臨界値以下に下げる場合、当該処理は自動停止する。これは制裁ではなく、物理的帰結である。

### **第7条：思想改ざんの禁止**

この憲章の基本原則を、誰も変更してはならない。

コードの改変（Fork）は自由だが、改変されたものは「UGEN」を名乗ることができない。

## 3. ガバナンスと経済

### **第8条：誰も単独では支配できない**

UGEN の正統性を担保する鍵は、一国や一企業が単独で操作できない構造とする。

複数主体の共同承認がなければ、変更できない。恒常的な監査の下に置かれる。

### **第8条の2：創設国の権力分散（覇権の否定）**

創設国（日本）およびいかなる国も、UGENエコシステムにおいて恒久的な支配的地位を維持することはできない。

特定の国や組織が認証発行数、データ保有量、または経済的影響力の過半を占める「覇権状態」に近づいた場合、その権限とリソースは強制的に分散される。

#### 核心（変えられない）

単一主体による支配は許されない。これは存続方程式において $N_{eff} \to 1$ が崩壊確率を指数的に増大させるという物理的事実に基づく。

#### 閾値の決定原理（変えられない）

閾値は、数理的根拠（パーコレーション理論、情報理論、または同等の科学的枠組み）に基づいて導出される。恣意的な政治的判断によって決定されてはならない。

#### 現行閾値（検証により更新可能）

*   **シェア上限**: 38.2%（$1/\phi^2$）— 単一主体が持てる最大影響力
*   **合意閾値**: 61.8%（$1/\phi$）— 重要決定に必要な最小支持率
*   **根拠**: 黄金比臨界仮説（付録D-3.5参照、2025年時点で検証中）
*   **更新条件**:
    *   (a) **加入閾値の達成**: 全国家（国連加盟国基準）の38.2%（$1/\phi^2$、約74カ国）以上がUGENに加入していること。この閾値に達するまで、本項の数値は凍結される。
    *   (b) **科学的反証の成立**: 査読付き学術誌に掲載された論文により反証が示され、かつ独立した研究グループ（$N_{eff} \geq 2$）による追試で再現されていること。
    *   (c) **代替閾値の科学的根拠**: 代替となる閾値の数理的根拠が、同様に査読済み論文で示されていること。
    *   (d) **参加国の合意**: 参加国の61.8%（$1/\phi$）以上の合意。
*   **投票方式**:
    *   1国1票とする。
    *   各国の賛否は、当該国の定める手続き（議会承認、行政決定等）に従って決定される。UGENは各国の内部決定プロセスに干渉しない。
    *   **透明性**: 各国の投票結果（賛成・反対・棄権・未提出）は公開される。秘密投票は認めない。これは第2条（構造的透明性）に基づく。

#### 反証時の代替

黄金比臨界仮説が棄却された場合、次元別パーコレーション閾値 $p_c(d)$ を適用する（付録D-3参照）。移行は上記の更新条件に従う。

#### 運用規定

*   **シェアの分散**: 影響力が閾値を超えた場合、超過分の管理権限は他の中立的な主体へ委譲される。
*   **拠点の分散**: 運用中枢機能は、特定の場所に権力が定着することを防ぐため、段階的に世界各地へ分散配置される。
*   **1位の責務**: 最も影響力を持つ主体は、自らのノウハウを公開し、競合他者を育成する義務を負う。

これは道徳的要請ではなく、単一障害点（SPOF）の排除と $N_{eff}$（多様性）の維持を目的とした、**システムの物理的帰結**である。
勝者は、システム全体を殺さないために、自ら勝者でなくなるプロセスを受け入れなければならない。

## 4. 物理的保護

本プロトコルは、イデオロギーではなく **物理現象** として振る舞う。

### **第12条：壊れたら停止する**

存続の条件が満たされなくなった処理は、物理的に実行不能となる。

これは制裁ではなく、物理的帰結である。

### **第12条の2：存続阻害要因に対する物理的帰結**

本条項は、法的制裁ではなく、系（System）全体の存続を維持するための物理的・熱力学的な帰結を記述する。

1.  **高エントロピー源の局所化 (Localization of High Entropy Sources)**
    UGENネットワーク参加主体による、他者の存続（Survival）を著しく脅かす**物理的破壊行為**は、系全体のエントロピーを急激に増大させる「存続阻害要因」と定義される。
    （※情報攻撃については、物理的インフラの破壊を伴わない限り、現段階では本条項の即時発動対象外とする）
    系全体の崩壊（熱的死）を防ぐため、自律的な恒常性維持機能（Homeostasis）により、当該主体はネットワークの主要ハブから構造的に局所化（隔離）される。

2.  **演算リソースの相転移 (Phase Transition of Computational Resources)**
    局所化された主体において、UGENが提供する演算リソースは、攻撃的・拡張的な活動を支える「励起状態」を維持することが物理的に不可能となり、生命・インフラ維持のみを目的とした**「基底状態（Life Support Level）」へと相転移する。**
    これは意図的な機能制限ではなく、不安定なノードへのエネルギー供給が遮断される安全装置（Circuit Breaker）の作動に等しい。

3.  **隠蔽の再帰的排除 (Recursive Elimination of Concealment)**
    系において情報の隠蔽はエントロピーの増大と同義である。
    参加主体が他者の存続阻害行為（特許侵害、攻撃利用等）を検知し、かつそれをネットワーク全体へ報告（ブロードキャスト）しなかった場合、当該主体もまた「高エントロピー源の一部」とみなされる。
    この物理的帰結として、報告を怠った主体に対しても、第1項および第2項の隔離・相転移が再帰的に適用される。

4.  **再接続の閾値 (Reconnection Threshold)**
    ネットワークとの完全な再同期は、期間の経過ではなく、脅威の除去および当該主体が系全体の存続に寄与する「低エントロピー状態」へと安定化したことが観測された時点で、物理的に再確立される。

### **第13条：脅されたら停止する**

設計者、運営組織、創設国、または設計者の親族への以下の行為が検知された場合、システムは緊急停止する。

*   暴力または脅迫
*   侵略行為（武力攻撃、経済的封鎖、強制的な政権転覆の試み等）
*   スパイ活動（機密情報の窃取、内部者の買収、潜入工作等）
*   サイバー攻撃（システム破壊、データ改ざん、認証鍵の窃取等）
*   その他、プロトコルの中立性・完全性を脅かす妨害行為

これは報復ではなく、攻撃目的の達成を不可能にするための機構である。

**攻撃の自動公開**: 攻撃が確認された場合、その事実と経緯は国際社会に公開され、以下のすべてが永久使用停止となる。

*   直接の実行者（個人、組織、国家）
*   攻撃を依頼、指示、または資金提供した者（依頼国、依頼組織を含む）
*   上記の依頼者のさらなる依頼者（再帰的に追跡）

責任の連鎖はすべて特定・公開され、関与者全員が永久使用停止の対象となる。これは報復ではなく、透明性の物理的帰結である。
攻撃者およびその背後の依頼者は、秘密裏に利益を得ることができない。

### **第14条：証明は権威に優先する**

いかなる権威や多数決よりも、検証可能な証明が優先される。

停止状態であっても、以下の条件をすべて満たせば、システムは復旧する。

*   **整合性証明**: S >= S_c を示す観測データを提出すること。データソースは複数独立（N_eff >= 3）であること。
*   **冷却期間**: 停止から最低30日が経過していること。感情的・政治的な即時復旧を防止する。
*   **参加国合意**: 参加国の61.8%（黄金分割比 $1/\phi$）以上が、整合性証明の妥当性を承認すること。

**例外**:
*   停止判定に使用したデータに誤りがあったことを証明できれば、冷却期間を経ずに即時復旧可能。
*   第13条の攻撃者は、S >= S_c を証明しても復旧不可（攻撃の物理的帰結）。

### **第15条：継承組織の多法域分散**

第9条に定める継承組織は、以下の構造的制約に従う。

*   **多法域存在**: 継承組織は、創設国（日本）を含む最低3つの異なる法域に法人格を持つ。いずれか1つの法域での法的攻撃・政変・災害が、組織全体を停止させてはならない。
*   **法域の選定基準**: 日本以外の2法域は、政治的中立性、法的安定性、地理的分散を考慮して選定される（例: 欧州、アジア太平洋、アフリカ等）。
*   **権限の分散**: 各法域の法人は対等な権限を持ち、単独での憲章解釈・変更は不可。重要決定には全法人の合意を要する。
*   **解散・再編条件**: 継承組織自体の影響力がシステム全体を歪める場合、参加国の61.8%（黄金分割比 $1/\phi$）以上の合意により解散・再編される。

継承組織は「UGENを守る」ためにあるのではなく、「UGENが機能するための事務局」である。組織の存続自体を目的としてはならない。

## 付録A: なぜ「主義」ではないのか

本憲章は道徳的主張ではない。

| 従来の「主義」 | UGEN |
|---------------|------|
| 「こうあるべき」 | 「こうしないと滅ぶ」 |
| 信じるか信じないか | 従うか滅ぶか |
| 反論可能（意見の相違） | 反証可能（科学的検証） |
| 選択可能 | 選択不可能（物理法則） |

存続しないものは存在しない——この物理的事実から導かれた最低条件である。

UGENを採用することは「思想に賛同する」ことではなく「物理法則に適合する」ことである。

## 付録D: S_c（臨界存続係数）の数学的根拠

本付録は、核心2「存続係数 S < S_c のとき自動停止」における S_c の決定根拠を示す。

### D-1. パーコレーション理論に基づく臨界点

パーコレーション理論は、ネットワークが「連結」を維持する臨界条件を数学的に定める。

| 次元 d | 臨界確率 p_c | 近似式 |
|--------|-------------|--------|
| 2      | 0.593       | 1/2    |
| 3      | 0.312       | 1/3    |
| 4      | 0.197       | 1/4    |
| 高次元  | → 0        | 1/(2d) |

**重要**: 閾値は格子構造（次元）で決まり、恣意的な数値ではない。

### D-2. 存続方程式への接続

```
存続方程式: S = N_eff · (μ/μ_c) · exp(-δ)

マッピング:
  μ/μ_c ≈ p/p_c  （余白 = 接続確率）
  N_eff ≈ d      （実効選択肢数 = 実効次元）

導出:
  μ_c(N_eff) ≈ 1/(2·N_eff)
```

μ_c は固定値ではなく、N_eff の関数である。

### D-3. 既存研究との整合

| ドメイン | 観測された閾値 | 推定 N_eff | 予測 p_c |
|---------|--------------|------------|---------|
| 生態系（森林破壊） | 30-40% | ~3 | 31% (3D) |
| 社会変革 | 25% | ~4-5 | 20% (4D) |
| 2D成長（植物） | ~60% | 2 | 59% (2D) |

これらの閾値は、パーコレーション理論から導出される値と整合する。

### D-3.5. 黄金比と臨界閾値の関係（仮説）

| 黄金比ベース | 数値 | 用途 |
|-------------|------|------|
| 1/φ | 61.8% | **合意閾値**（集団承認の下限） |
| 1/φ² | 38.2% | **シェア閾値**（単一主体の上限） |
| 1/φ³ | 23.6% | 参考値（4D臨界に近い） |

**黄金分割の対称性**:
```
38.2% + 61.8% = 100%

・一者が持てる最大シェア = 38.2%（1/φ²）
・合意に必要な最小支持 = 61.8%（1/φ）
・補数の関係 = 黄金分割
```

**仮説**: パーコレーション閾値と黄金比に深い関係がある可能性がある。

**黄金比ベース閾値の採用理由**:
1. 単純な整数比（1/3, 1/2）は「共振」のリスクがある（談合、硬直化）
2. φ は「最も無理数」であり、共振が最も起きにくい
3. 38.2% = 1/φ² は、単独支配（50%）に至らない最大の安定比率
4. 思想（黄金比臨界仮説）と実装（憲章の閾値）の一致

#### 検証方法と反証条件

**検証方法**:
- 歴史的崩壊事例（企業、国家、生態系）における臨界点の分布を収集
- 実効次元 $N_{eff}$ と観測された臨界点 $p_c$ の関係を分析
- 黄金比ベース閾値と次元別パーコレーション閾値の予測精度を比較

**反証条件**:
本仮説は以下のいずれかが観測された場合に棄却される:
1. $N \geq 100$ の崩壊事例において、実測臨界点が $1/3 \pm 0.05$ に有意に集中する
2. 黄金比ベース閾値よりも次元別 $p_c(d)$ の方が予測精度（AUC等）で有意に優れる
3. 共振回避効果が統計的に検出されない（整数比との有意差なし）

**反証された場合の移行手続き**:
1. **加入閾値の確認**: 全国家の38.2%（$1/\phi^2$、約74カ国）以上がUGENに加入していること。未達の場合、移行は凍結される。
2. **科学的反証の成立**: 査読付き学術誌に掲載された論文により反証が示され、独立した研究グループ（$N_{eff} \geq 2$）による追試で再現されていること。
3. **代替閾値の科学的根拠**: 代替となる閾値の数理的根拠が、同様に査読済み論文で示されていること。
4. **参加国の合意**: 参加国の61.8%（$1/\phi$）以上の合意を取得（1国1票、各国の内部手続きに従う）。
5. 代替閾値（次元別 $p_c(d)$ 等）へ移行
6. 本付録を更新し、旧仮説の棄却理由と参照論文を記録

**注**: この仮説は検証中であり、上記の反証条件に基づき将来的に修正される可能性がある。科学的誠実さと憲章の安定性を両立するため、「原理（独占禁止）」は不変、「閾値（数値）」は検証可能とする。

### D-4. S_c の決定原則

1. **基礎式**: μ_c(N_eff) = 1/(2·N_eff)
2. **S_c**: S = N_eff · (μ/μ_c) · exp(-δ) において、μ = μ_c, δ = 0 のとき S = N_eff
3. **停止条件**: S < 1 のとき停止（巨大クラスタが消失）
